Usage

Below are swatches of the entire color palette. They should always be accessed using the function palette($color, $tone). The classes in the example HTML are for styleguide purposes only.

Example
// Set a variable to use in your styles
$my-variable: palette(secondary, dark);

// Or access colors from the palette directly
.my-component {
  background: palette(primary, base);
  color: palette(greyscale-cool, x-light);
}
